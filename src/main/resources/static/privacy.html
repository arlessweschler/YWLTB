Information Collection and Use

The app does not collect or store any personal information, including names, emails or addresses. The information the app collects and stores is your Telegram user ID and YouTube API access token.

Telegram user ID is used to associate you Telegram account and messages you send to the app via Telegram with your YouTube API token.

YouTube API token is used to add videos to your "Watch later" playlist.

Both Telegram user ID and YouTube API access token are stored after you configure the app by linking your Telegram account with your YouTube API access token by providing this app access to manage your YouTube content.

You can revoke the access to your YouTube content at any time by removing the app from the list of the apps with access to your account in your Google Profile settings.

The app does not use any third party services that may collect information used to identify you.
